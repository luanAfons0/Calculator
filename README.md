# Calculator

## Overview
This project is a **back-to-basics calculator** built to reinforce core frontend concepts while learning modern React development. It was created as part of a hands-on learning experience using the course linked below.

The main goal is to practice fundamentals such as component structure, state management, and TypeScript integration in a simple and focused project.

## Learning Reference
This project is based on concepts and guidance from the following course:
